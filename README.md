🌱 Eco Nest – Eco-Friendly Products Mobile App 📱

Eco Nest is an Android application promoting sustainable consumption in line with UN SDG 12 (Responsible Consumption & Production). This project demonstrates a complete mobile app with user authentication, profile management, navigation, and interactive features like an eco-quiz.

🔹 Features
User authentication and profile management 🧑‍💻
Navigation between app sections with smooth UI/UX 🛠️
Local data storage using SQLite and Shared Preferences 💾
Eco-quiz to encourage sustainable habits 🌎
Designed for scalability and offline support ⚡

🔹 Tech Stack
Frontend: Flutter / Android Studio
Database: SQLite (local storage)
Authentication & Storage: Shared Preferences
Other Tools: Git for version control

🔹 Installation & Setup
Clone the repository:
git clone https://github.com/PabasaraSenevirathne/EcoNest.git
Open in Android Studio.
Run the app on an emulator or real device.
Enjoy exploring eco-friendly features! 🌱
